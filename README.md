Introduction
============

Download kubectl and install into /usr/local/bin/kubectl.

How to use
----------

Run the following commands in Linux console.

    git clone https://github.com/liangsuno/setup-kubernetes.git
    cd setup-kubernetes
    sudo yum install make
    make
